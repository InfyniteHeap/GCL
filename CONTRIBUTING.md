# Contributing to Grid Craft Launcher

Thank you for your interest in contributing to Grid Craft Launcher!

## Build This Project

Before getting start, you **MUST** install these components:

- [Rust](https://www.rust-lang.org/)
- [Visual Studio](https://visualstudio.microsoft.com/)

You must also add **Windows application development** workload for your Visual Studio, or this app cannot be built!

To build the app in development mode, open the sln file in Visual Studio (Grid Craft Launcher.sln), then you can build and debug this app!
